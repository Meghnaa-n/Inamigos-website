# InAmigos Foundation Website

A responsive NGO awareness website developed for **InAmigos Foundation** to showcase its mission, projects, impact, volunteer opportunities, and community initiatives.

## About

This website aims to spread awareness about the work of InAmigos Foundation and encourage community participation through volunteering and social engagement.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and user-friendly interface
- About Us section
- Projects and Initiatives showcase
- Impact Statistics section
- Volunteer Registration section
- Gallery section
- Contact section
- Smooth scrolling and interactive UI elements

## Technologies Used

- HTML5
- CSS3
- JavaScript
